# Event Images Directory

Add your OpenMind event images here:

- `event1.jpg` - OpenMind Summit 2025 - Stage & Podium
- `event2.jpg` - OpenMind Team Leaders & Networking
- `event3.jpg` - OpenMind Speaker Presentation
- `event4.jpg` - OpenMind Keynote & Audience
- `event5.jpg` - OpenMind Conference & Attendees

## Specifications
- Format: JPG or PNG
- Recommended size: 800x500px or larger
- Quality: High resolution (85%+ quality)
- Aspect ratio: 16:9 (landscape)

## How to Add Images
1. Place your event images in this directory
2. Name them `event1.jpg` through `event5.jpg`
3. Images will automatically load on the homepage "Previous Events" section
4. If images are not found, fallback images from Unsplash will be displayed

## Notes
- Images are automatically optimized with brightness and contrast enhancement
- Hover effect includes a slight zoom animation
- Lazy loading is enabled for performance
