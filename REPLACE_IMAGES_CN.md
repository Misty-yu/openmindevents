# 如何替换"Our Previous Events"图片

## 方法一：通过媒体库上传（推荐）

### 步骤：

1. **点击导航栏 → 媒体库** （或直接访问 `/admin/media`）

2. **选择"活动图片"标签页**

3. **在"活动图片"标签下，选择"往期活动图片"子标签**

4. **点击虚线框上传您的图片**
   - 支持 JPG, PNG, WebP, GIF 格式
   - 最大 10MB

5. **图片会自动显示在首页轮播区域**

---

## 工作原理

- 您上传的图片存储在 Supabase Storage 的 `event-images` 存储桶
- 特别是在 `past-events` 文件夹中
- 首页组件会自动从该文件夹加载图片
- 如果该文件夹没有图片，会显示默认的占位图片

---

## 删除已上传的图片

1. 进入 `/admin/media`
2. 在"往期活动图片"图库中
3. 鼠标悬停在要删除的图片上
4. 点击红色的删除按钮（垃圾桶图标）
5. 确认删除

---

## 更换图片

### 方法A：直接更换
1. 删除旧图片
2. 上传新图片

### 方法B：覆盖上传
- 新上传的图片会自动添加到图库
- 删除不需要的旧图片即可

---

## 图片显示顺序

- 按上传时间倒序排列（最新上传的在前）
- 要改变顺序，可以删除后重新上传

---

## 图片要求

| 属性 | 建议 |
|------|------|
| 格式 | JPG, PNG, WebP, GIF |
| 大小 | < 10MB（建议 500KB - 2MB） |
| 尺寸 | 800×500 或 16:10 比例 |
| 数量 | 建议 5-10 张 |

---

## 故障排除

### 图片没有显示？

1. **检查文件夹**：确保上传到了 `past-events` 文件夹
2. **刷新页面**：上传后刷新首页
3. **检查格式**：确保是 JPG/PNG/WebP/GIF 格式
4. **查看图库**：在媒体库确认图片已上传成功

### 显示的是默认图片？

- 这意味着 `past-events` 文件夹中还没有图片
- 上传您的第一张图片后，就会自动替换

### 轮播速度太慢/快？

可以联系开发团队调整 `past-events.tsx` 文件中的 `speed` 参数：

```typescript
const speed = 0.5; // 数值越小越慢，越大越快
```

## 直接使用URL

如果您想在某张图片的显示URL：

1. 在媒体库中，点击"选择"按钮
2. URL会显示在页面顶部
3. 复制使用

## 高级用法

### 批量上传

如果您想一次性上传多张图片，可以：

1. 使用编程方式：

```typescript
import { uploadFiles } from '@/lib/storage';

const files = [file1, file2, file3];
const results = await uploadFiles(files, 'event-images', 'past-events');
```

2. 或者逐个上传（推荐）

### 自定义图片描述

图片的 alt 文本是自动从文件名生成的：
- 文件名：`my-conference-2025.jpg`
- 显示为：`my conference 2025`

建议在上传前重命名文件为有意义的名称。
